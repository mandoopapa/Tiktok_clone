### 설치하기

```
git clone https://github.com/mandoopapa/Tiktok_clone

cd Tiktok_clone

flutter pub get

flutter run
```

iOS Simulator, Android Simulator로 구동이 가능합니다.

Xcode 설정을 통해 직접 스마트폰에서 앱을 실행해보세요!
